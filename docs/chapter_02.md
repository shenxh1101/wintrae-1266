# 第2章 古村探秘

经过三个小时的山路跋涉，李明轩、苏婉清和王大勇终于到达了地图上标记的古村。

村子坐落在**苍梧山脉**的半山腰，房屋都是用当地的石头垒砌而成，屋顶覆盖着厚厚的青苔。

「这个村子看起来至少有五百年历史了。」苏小姐推了推眼镜，兴奋地打量着四周。

老王点了点头：「我听说这里叫『守玉村』，村民世代守护着什么东西。不过近十年来，年轻人都搬走了，只剩下几个老人。」

## 初入古村

村口坐着一位白发苍苍的老人，正抽着旱烟。看到三人走近，老人抬起头，浑浊的眼睛里闪过一丝警惕。

「你们找谁？」

「老人家，我们是古韵博物院的，想打听一些关于**玄玉**的事情。」婉清礼貌地说道。

听到「玄玉」两个字，老人的脸色瞬间变了。他站起身，转身就往村里走，嘴里念叨着：「不知道，不知道……」

## 神秘的祠堂

李明轩注意到村子中央有一座格外气派的祠堂，屋顶的瓦片泛着奇异的光泽。

「王先生，你看那座祠堂。」明轩指着前方说道。

王大勇眯起眼睛：「奇怪，我三十年前来过这里，记得当时那座祠堂是锁着的，从不对外开放。」

三人走近祠堂，发现大门虚掩着。苏婉清轻轻推开门，里面的景象让所有人都屏住了呼吸。

![ancient_shrine](./images/shrine.jpg)

祠堂正中供奉着一尊巨大的玉雕，雕刻的是一条盘旋于云端的巨龙。巨龙的双眼镶嵌着两颗深蓝色的宝石，在昏暗的光线下散发着幽光。

「这是……**龙渊**的图腾？」婉清声音颤抖着说道。

## 神秘的祠堂

祠堂的墙壁上绘满了壁画，讲述着一个古老的故事：

- 上古时期，一颗陨石降落在苍梧山中
- 陨石中蕴含着神秘的星辰之力
- 先民们用陨石打造了两把神剑和一枚玉佩
- 玉佩就是**九天玄玉**，两把神剑分别名为「龙渊」和「虎魄」

李明轩仔细观察着壁画，发现最后一幅画描绘的是一群人将玄玉和其中一把剑藏在了山洞中，山洞的入口处有一个独特的标记——三棵呈品字形排列的古松。

## 守玉人的后人

天色渐晚，三人决定在村里唯一的民宿过夜。民宿的老板娘是个四十多岁的妇女，自称是守玉人的后代。

「你们要找的东西，就在后山的山洞里。」老板娘压低声音说道，「不过我劝你们别去，那个地方……不干净。」

「不干净？」李先生疑惑地问道。

老板娘打了个寒颤：「三十年前，有一群人不听劝，硬要进去。结果只有一个人活着出来，出来以后就疯了，嘴里一直念叨着『星星醒了』。」

苏婉清和明轩对视一眼，都看到了对方眼中的坚定。无论前方有什么危险，他们都必须找到玄玉。
